Dewslider diaporama with CCK Image Field

Provide CCK Image Field formatters for Dewslider.

== Installation ==
Download dewslider from http://www.alsacreations.fr/dewslider.html.
Copy dewslider.swf in imagefield_dewslider/dewslider.
Install the module.

== Configuration ==
Choose Dewslider as formatter for a imagefield field.
You can modify dewslider settings from admin/settings/imagefield_dewslider.
